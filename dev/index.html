stlcsb

The stlcsb package enables the tidy and reproducible analysis of 311 data from the City of Saint Louis. It's a crucial element of ongoing research. On CRAN and GitHub

censusxy

The censusxy package brings a crucial source of open geocoding to the R community. This package allows for free and virtually unlimited geocoding of addresses in the United States. On GitHub

gateway

The creation of Christopher Prener, Ph.D., gateway is the solution to working with the City of St. Louis's open data in R. My contribution is assisting in the creation of a composite geocoder for the St. Louis region. On GitHub
