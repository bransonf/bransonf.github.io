This is the result of some experimentation with the D3 javaScript library. Currently, it is used to draw SVG paths from geometry and define a fill based on an associated value. This will eventually be modified to use texture fill.

R ggiraph

ggplot2 is among the most notorious R visualization packages, and ggiraph allows that to extend to some interesting web visualizations. In this simple example, tool tips are added to a choropleth map.
